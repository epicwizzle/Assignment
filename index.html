Group 1: Question 2

Read the input from the document and if it is a positive number interpret it as some number of days. Convert it into years, months, days. To make things easier, assume there is 365.25 days per year and that every month is 30.44 days. Left over days output may have a decimal portion, but years and months should be integers. Output these values with appropriate labels. If the input is not a number output “Non-numeric or negative input”.

Note: decimal math is prone to slight rounding errors, don’t be concerned about an answer like 14.779999999 vs 14.78 if you see it. For added challenge use Math.round to guarantee 2 decimal places.

Example: input: 2175.87 output: 5 years, 11 months, and 14.78 days
